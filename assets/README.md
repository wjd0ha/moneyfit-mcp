# MoneyFit Image Assets

PlayMCP registration requires a representative image of at least 600x600 pixels.

## Recommended

- `moneyfit-logo-source-600.png`
  - 600x600 PNG
  - Cropped from the approved MoneyFit brand board.
  - Recommended for PlayMCP representative image because it preserves the original symbol, Korean logo, English tagline, and slogan.

## Alternatives

- `moneyfit-icon-600.png`
  - 600x600 PNG
  - Symbol-only app icon style.
  - Useful when the image is displayed very small.

- `moneyfit-logo-600.png`
  - 600x600 PNG
  - Recreated logo layout generated from `scripts/generate_moneyfit_logo.py`.
  - Useful as an editable/generated fallback.

## Brand Colors

- Navy: `#0B1D3A`
- Green: `#11865B`
- Gold: `#D4AF37`
